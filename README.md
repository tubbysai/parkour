# Parkour Arcade Games

## Ice Slide

Ice Slide is available as a self-contained HTML game at
[`ice-slide.html`](./ice-slide.html). GitHub Pages visitors can also use the
clean [`/ice-slide/`](./ice-slide/) route, which forwards to the same game file.

No web server, package installation, or build step is required. To play
locally, open `ice-slide.html` directly in a modern browser.
